## How to Contribute

Interested in contributing? Please see the [issues](https://github.com/j3-signalroom/apache_flink-kickstarter-ii/issues) page for open tasks, or open a new issue to start a discussion.

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes
4. Push to your fork and open a Pull Request

Contributors will be added to this file as contributions are merged.
